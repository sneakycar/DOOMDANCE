Drop your night alley panos here (PNG or JPG).

Each file can be:
  • One wide horizontal painting → auto-sliced every 480px at 270px tall
  • One image already ~480×270 → resized to fit one screen

When you have files in this folder, run from godot-philly-alley:

  ./tools/import_painted_segments.sh

Imported segments land in:
  assets/segments/painted/night/

Then press F5 in Godot.
